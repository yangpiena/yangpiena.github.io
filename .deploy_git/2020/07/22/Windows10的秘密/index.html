设置任务栏透明效果

  1. 设置 -> 颜色 -> 打开“透明效果”开关
  2. 打开注册表，找到路径：
  3. 右侧找到名称 TaskbarAcrylicOpacity，如果没有，就右键新建一个DWORD（32位）值D
  4. 将值修改为十进制 0 就是全透明，也可以是十进制0-10之间的效果
  5. 不用重启电脑，在任务管理器重启“windows资源管理器”就生效
