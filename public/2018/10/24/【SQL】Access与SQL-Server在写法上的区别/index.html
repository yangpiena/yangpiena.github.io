8. Access类似于SQL Server的函数ISNULL()

9. Access创建表或添加列时，必须附带默认约束（Defualt）

10. Access创建新列时，如果未添加默认约束，在采用内置函数STR(A)查询时会显示null值

如下：

ALTER TABLE 表 ADD A VARCHAR(23)
SELECT IIF(STR(A) IS NULL, '空', '非空') FROM 表
